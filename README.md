# CastleVania
Castlevania Clone made in C++ for a college project

Portugues: 

Este projeto foi feito para conclusão da matéria "Técnicas de programação" do curso de Engenharia de Computação da UTFPR/Curitiba, em 1/Sem de 2023. Ele contem um diagrama de classes do que foi feito além dos arquivos necessários para compilar. 

English:

This project was made to complete the "Programming Techniques" subject of the Computer Engineering course at UTFPR/Curitiba, on the 1/Semester of 2023. It contains a class diagram of what was done in addition to the files needed to compile.
